## Проект 1: Прогнозирование стоимости недвижимости (House Prices)

Проект посвящен детальному анализу данных (EDA), предобработке признаков и построению линейной модели регрессии для прогнозирования стоимости жилых домов в городе Эймс (штат Айова) на основе набора данных [House Prices: Advanced Regression Techniques](https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques).

### Основные этапы работы:
* **Анализ структуры и очистка данных:** 
  * Проведена оценка пропущенных значений, визуализированы пропуски на тепловой карте.
  * Реализовано заполнение пропусков с использованием медианных значений по группам (например, `LotFrontage` по конфигурации участка `LotConfig`), моды для категориальных признаков и логических заглушек (например, `NoGarage`, `NoBasement`).
  * Удалены малоинформативные колонки и дублирующие площади для минимизации мультиколлинеарности.
* **Математическая статистика и анализ распределений:**
  * Рассчитаны основные описательные метрики числовых признаков: дисперсия (variance), асимметрия (skewness), эксцесс (kurtosis) и перцентили (от 5-го до 95-го).
  * Выполнена проверка на выбросы по методу трех сигм ($3\sigma$). На основе этого выявлен и удален аномальный объект (нетипичная частичная сделка с огромным подвалом по заниженной цене).
* **Feature Engineering & Encoding:**
  * Создан новый признак — возраст дома (`HouseAge`).
  * Произведено кодирование порядковых признаков качества (Quality Mapping) и бинаризация качественных признаков с помощью One-Hot Encoding.
  * Осуществлен отбор наиболее значимых признаков по уровню линейной корреляции с целевой переменной.
* **Моделирование и оценка качества:**
  * Построена линейная модель с L1-регуляризацией (**Lasso Regression**). В качестве целевой переменной использовался логарифм цены $\ln(1 + SalePrice)$.
  * Модель показала высокую объясняющую способность (коэффициент детерминации $R^2 \approx 0.90$ на валидационной выборке).
  * Проведен анализ стандартизированных и исходных весов (полуэластичности) для оценки влияния физических характеристик и районов города на конечную стоимость недвижимости.
